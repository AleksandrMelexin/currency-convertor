### Задание
**Тема**: Курсы валют / Конвертер

**Описание**:  
Создайте SPA приложение, которое показывает текущие курсы валют и позволяет конвертировать одну валюту в другую.

Вы можете использовать любой из следующих открытых API:
- [ExchangeRate.host](https://exchangerate.host/#/lookup) 
- [Open Exchange Rates](https://openexchangerates.org/) 
- [Free Currency Converter API](https://www.currencyconverterapi.com/) 

> Для большинства API требуется регистрация и получение API-ключа.

**Функционал**:
- Отображение списка валют с возможностью обновления
- Поле ввода для конвертации:
  - Выбор исходной валюты
  - Ввод суммы
  - Выбор целевой валюты
  - Отображение результата
- Возможность сохранения истории конвертаций
- Возможность просмотра графика изменения курса за последние 7 дней (mock или реальный API)

**Архитектура**:
- Использование Vue 3 (Composition API)
- Использование TypeScript
- Хранение состояния через **Pinia**
- Разделение на компоненты (например, `CurrencyConverter`, `CurrencyList`, `HistoryLog`)
- Адаптивный дизайн (Vuetify обязателен)

**Интерфейс**:
- Минималистичный, но удобный UI
- Обработка ошибок (ошибка API, неверные данные)
- Лоадеры/индикаторы загрузки данных

**Дополнительно (по желанию)**:
- График динамики курса (можно использовать chart.js или v-charts)
- Поддержка тем (светлая/темная)
- Автоматическое обновление курсов каждые N минут
  

**Что мы проверяем**:
- Понимание Vue 3 и Composition API
- Умение работать с внешними API
- Навык разделения приложения на компоненты
- Использование Pinia для управления состоянием
- Удобство интерфейса и адаптивность
- Чистота и структурированность кода
