# Конвертор валют

Приложение для конвертации валют, позволяет конвертировать валюту и просматривать историю совершённых операций

## Стэк технологий в FrontEnd части

- **Vue + Typescript + vuetify** 
- **Cборщик проекта: Vite**
- **State-менеджер: Pinia**
- **Архитектура проекта: FSD**

## Стэк технологий в BackEnd части

- **Node.js + Express + Typescript** 

### Почему решил написать серверную часть самостоятельно

Во время решения тестового задания рассмотрел API для работы с валютой, однако, в [ExchangeRate.host](https://exchangerate.host/#/lookup) бесплатный план предоставляет всего 100 запросов в месяц, чего не хватило бы для разработки. 
У [Open Exchange Rates](https://openexchangerates.org/) бесплатный тарифный план предоставляет 1000 запросов, однако имеет очень ограниченный функционал, я смог только посмотреть список валют и список их курсов. Эти данные я использовал в качестве моковых на сервере.
Сервис [Free Currency Converter API](https://www.currencyconverterapi.com/) в целом [приостановил](https://free.currencyconverterapi.com/) бесплатный тарифный план на неопределённый срок. В связи с этим я решил написать сервер самостоятельно чтобы продемонстрировать свою работу с API.

## Инструкция по запуску

1. Перейдите в корневую директорию проекта:
	```bash
	cd currency-convertor
	```
2. Соберите проект: 
	```bash
	docker-compose build
	``` 
3. Запустите проект:
	```bash
	docker-compose up -d
	```

Веб-приложение будет доступно по адресу: http://localhost:5173

Клиент будет запущен на порту 5173

Сервер будет запущен на порту 8080
